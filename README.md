# 🚌 AI-Based Integrated Bus Tracking System (AIBTS)

## 📌 Overview
A full-stack web application designed to improve public transport efficiency through real-time bus tracking, route management, and ticket booking. The system supports multiple user roles including passengers, drivers, and administrators.

---

## 🎯 Features
- 📍 Real-time bus tracking
- 🎫 Online ticket booking system
- 👨‍✈️ Driver dashboard for route updates
- 🛠 Admin panel for managing routes and buses
- 📊 Data insights for better transport planning

---

## 🧠 Tech Stack
- Frontend: Next.js, React, TypeScript
- Backend: Node.js / API Routes
- Database: MySQL

---

## ⚙️ System Architecture
- Multi-role authentication system
- REST API-based communication
- Database-driven route and ticket management

---

## 🚀 How to Run
1. Clone the repository  
2. Install dependencies using `npm install`  
3. Run the project using `npm run dev`
